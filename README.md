# course-spz-process-manager

Курсовий проєкт з дисципліни «Системне програмне забезпечення». Розробка програмного забезпечення для захисту від вірусів та шкідливих програм.

## Architecture

- `src/` — application source code.
- `docs/` — project documentation.
- `docs/diagrams/` — architecture and flow diagrams.
